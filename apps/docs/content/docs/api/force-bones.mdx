---
title: forceBones
description: Sentinel value for forcing skeleton mode on individual components.
---

```tsx
import { forceBones } from "bones";
```

`forceBones` is a frozen empty object typed as `Promise<never>`. When passed as data to `createBones`, it triggers skeleton mode for that specific component without needing a Suspense boundary or `<BonesForce>` wrapper.

## Usage

```tsx
import { createBones, forceBones } from "bones";

function UserCard({ user }: { user: Promise<User> }) {
  const { bone, data } = createBones(user);
  return (
    <div>
      <h2 {...bone("text", { length: 12 })}>{data?.name}</h2>
    </div>
  );
}

// Force skeleton mode for a single component:
<UserCard user={forceBones} />
```

## How it works

`createBones` checks if the input is the `forceBones` sentinel by identity comparison (`===`). When matched, it sets `isLoading = true` and `data = undefined` without throwing a promise or consulting the loading context.

Since `forceBones` is typed as `Promise<never>`, it's assignable to any `Promise<T>` prop.

## When to use

- When you need to force skeleton mode on **one specific component** rather than an entire subtree (use `<BonesForce>` for that).
- Used internally by `<Bones>` to swap promise props in the Suspense fallback tree.
